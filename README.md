# codex 自动化训练

这个目录用于沉淀每次 60 分钟以内的开发者基本功练习。

## 使用方式

1. 每次先读当天文件夹里的 `lesson.md`。
2. 按任务做一遍，不追求一次全记住，追求能独立复现。
3. 做完后，在同一文件夹放一个 `.txt` 阅读感受或卡点记录，例如 `feedback.txt`。
4. 下次自动化会优先看这些 `.txt`，再决定练习主题、难度和讲法。
